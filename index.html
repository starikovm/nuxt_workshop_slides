Build your component

  1. Run yarn dev
  2. Add button and two <p> to template
  3. Add data property to the component to store boolean
  4. Add methods property and create method for click
  5. Append @click to button and assign click method that will show first <p> on click
  6. Add computed property that will change second text upon first <p> text is shown

Vue.js got steroids

So what is that?

Nuxt.js includes the following to create a rich web application development:

  • Vue 2
  • Vue Router
  • Vuex (included only when using the store option)
  • Vue Server Renderer (excluded when using mode: 'spa')
  • vue-meta

A total of only 57kB min+gzip (60kB with Vuex). Under the hood there are webpack with vue-loader and babel-loader to bundle, code-split and minify your code.
